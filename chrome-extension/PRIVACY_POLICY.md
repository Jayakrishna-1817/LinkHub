# Privacy Policy for Smart Bookmark Organizer

**Last Updated**: December 19, 2025

## Overview

Smart Bookmark Organizer ("the Extension") is committed to protecting your privacy. This privacy policy explains how the Extension handles your data.

## Data Collection

**We DO NOT collect, store, or transmit any personal data to external servers.**

### What Data is Processed Locally

The Extension processes the following data **locally on your device only**:

1. **Bookmark URLs and Titles**
   - Used to classify and organize bookmarks
   - Analyzed using keyword matching algorithms
   - Never sent to external servers

2. **User Settings**
   - Stored locally using Chrome's storage API
   - Includes preferences like auto-organize on/off, notification settings
   - Synced via Chrome Sync if you have it enabled (Google's service)

3. **Usage Statistics**
   - Total bookmarks organized
   - Category usage counts
   - Stored locally on your device only
   - Used only for display in the extension popup

4. **Page Content (Optional)**
   - Only if you enable "Use page content for classification" feature
   - Analyzed locally for better categorization
   - Never stored or transmitted

## Permissions Explanation

### Bookmarks Permission
- **Why**: To read, create, and organize bookmark folders
- **What**: Access to your Chrome bookmarks
- **Usage**: Automatically categorize and move bookmarks to organized folders

### Storage Permission
- **Why**: To save your settings and statistics
- **What**: Local storage on your device
- **Usage**: Remember your preferences between browser sessions

### Tabs Permission (Optional)
- **Why**: To analyze active page content for better categorization
- **What**: Access to current tab's URL and content
- **Usage**: Only when "Use page content" feature is enabled
- **Note**: Content is analyzed locally and immediately discarded

### Notifications Permission
- **Why**: To inform you when bookmarks are organized
- **What**: Ability to show browser notifications
- **Usage**: Display success messages when auto-organizing

## Data Storage

- **Location**: All data is stored locally on your device using Chrome's storage API
- **Sync**: If you have Chrome Sync enabled, your settings may sync across your devices via Google's servers (Google's privacy policy applies)
- **Deletion**: Uninstalling the extension removes all stored data

## Third-Party Services

**We do not use any third-party services, analytics, or tracking.**

- No Google Analytics
- No advertising networks
- No crash reporting services
- No external APIs (except Chrome's built-in APIs)

## Data Security

- All processing happens locally on your device
- No data transmission to external servers
- No user accounts or authentication required
- No personal information collected

## Children's Privacy

The Extension does not knowingly collect any information from children under 13. The Extension is designed to be used by anyone capable of using Chrome browser.
