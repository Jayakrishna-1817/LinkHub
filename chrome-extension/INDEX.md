## 📞 Getting Help

### Self-Help Resources (In Order)

- What you tried
- Error messages
- Console logs
- Steps to reproduce

## ✅ Quick Checklist

### Installation
- [ ] Icons created (4 PNG files)
- [ ] Extension loaded in Chrome
- [ ] No error messages
- [ ] Icon visible in toolbar
- [ ] Service worker active

### Testing
- [ ] Bookmarked Instagram → Social Media folder
- [ ] Bookmarked YouTube → Entertainment folder
- [ ] Popup shows statistics
- [ ] Settings page opens
- [ ] Notifications appear

### Troubleshooting
- [ ] Read relevant docs
- [ ] Checked console logs
- [ ] Verified permissions
- [ ] Tested with clean bookmark
- [ ] Tried reload/restart
